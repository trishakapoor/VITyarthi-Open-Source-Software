## Project Description
This project is part of the Open Source Audit assignment. It analyzes Python as an open-source software and demonstrates Linux skills using shell scripting.

Five scripts are created to perform system monitoring, package checking, directory analysis, log analysis, and user interaction.

## Scripts Description

Script 1 — System Identity Report  
Displays Linux distribution, kernel version, user, home directory, uptime, date, and license.

Script 2 — Package Inspector  
Checks if Python is installed and shows its version.

Script 3 — Directory Auditor  
Analyzes directories like /etc, /home, and /usr/bin and shows size and permissions.

Script 4 — Log Analyzer  
Counts error entries from system log files.

Script 5 — Manifest Generator  
Takes user input and prints an open-source statement.
